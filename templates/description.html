Do machine learning models understand the core concepts of intuitive psychology? We present AGENT (Action, Goal, Efficiency, coNstraint, uTility), an empirically-grounded challenge consisting of a large dataset of procedurally generated 3D animations, which probes key concepts in intuitive psychology. There are 1920 training trials, 480 trials, and 960 trials. For each trial, a model needs to give a surprising score for a test video based on the corresponding familiarization video(s). Test trials are paired (a trial with the expected test video and a trial with the corresponding surpersing test video). For a pair of trials, if the model gives a lower surprising score to the expected test video than the surprising test video, we count it as correct.

BACKGROUND: For machine agents to successfully interact with humans in real-world settings, they will need to develop an understanding of human mental life. Intuitive psychology is the ability to reason about hidden mental variables that drive observable actions, and it comes naturally to people. Even babies can tell agents and objects apart, and they expect agents to act efficiently to achieve goals given constraints.

Despite recent interest in machine agents that reason about other agents, it is not clear if such agents learn or hold the core psychology principles that drive human reasoning. This dataset and challenge help to investigate these concepts, by testing whether a model can reason about the mental state of an agent by watching its actions in physical environments and judge whether its behavior in a new environment is consistent with its mental state and the new physical environment.
